# React / redux boilerplate project

## Tech Stack

* Language: Javascript ES6 + JSX
* View: React.js
* Application state management: Redux, react-redux
* Code check: eslint
* Unit test: enzyme, tape
* Packaging: webpack

## Build

```
$ npm run build
```

## Code check & unit test

```
$ npm t
```


## Install

```
$ npm install
```

## Run

```
$ npm start
```
Open browser on http://localhost:8080/
