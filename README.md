# React / Redux Starterkit

## Inside the box

| Category     | Name                  |
|--------------|-----------------------|
| Language     | JavaScript (ES6, JSX) |
| Libraries    | - React               |
|              | - Redux               |
| Code Quality | - Eslint              |
|              | - Airbnb Styleguide   |
| Testing      | - AVA                 |
|              | - Enzyme              |
| Packaging    | - Webpack             |

## How to use

### Install

```
$ yarn
```

### Run

```
$ yarn start
```

Open browser on [localhost:8080/](http://localhost:8080/)

### Code linting and unit testing

```
$ yarn test
```

### Build

```
$ yarn run build
```
